## 配置项目
#### 基本信息配置
进入src/utils文件夹,打开config.js文件编辑网站基本信息
```
const title = "网站标题";
const url = "后端地址";
const icp = "icp备案信息";
```

## 项目初始化
```
npm install
```
### 编译运行
```
npm run serve
```

### 编译打包
```
npm run build
```
